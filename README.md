# Royal Craft Construction

A single-page website template for **Royal Craft Construction** — a premium construction company. Dark theme with gold accents and clear sections for hero, about, services, projects, and contact.

## Contents

- **index.html** — Main page structure and content
- **styles.css** — Layout, typography (Cormorant Garamond + Outfit), and responsive styles
- **script.js** — Mobile menu, header scroll state, contact form demo, footer year

## Run locally

Open `index.html` in a browser, or use a local server:

```bash
npx serve .
```

Then open the URL shown (e.g. http://localhost:3000).

## Customization

- **Colors**: Edit CSS variables in `:root` in `styles.css`: `--color-primary` (Leo's Gold), `--color-secondary` (Virgo's Green), `--color-cta` (copper), `--color-accent` (charcoal), `--color-bg` (warm off-white).
- **Content**: Update text, phone, email, and address in `index.html`.
- **Images**: Replace the placeholder project/about areas with real images by adding `<img>` tags or background-image URLs and adjusting `.project-image` / `.about-image-frame` as needed.
